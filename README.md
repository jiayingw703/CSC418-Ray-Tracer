# CSC418-Ray-Tracer
Distributed ray tracer built for CSC418 assignment 3

# Features
The ray tracer has the following features:
* Handling non-trivial compound object
* Glossy refection
* Depth of field
* Soft shadows
* Texture mapping
* Refraction
* Multi-thread parallelization (OpenMP)
